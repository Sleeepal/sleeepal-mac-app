# Contributing

Thanks for contributing to Sleeepal Mac App.

## Before You Start

- Use macOS 14 or later
- Make sure Swift 6.2 is available
- Keep changes focused and easy to review

## Local Development

```bash
swift build
./Scripts/build-app.sh
```

## Pull Requests

- Create a branch for each change
- Keep commit messages short and descriptive
- Explain user-visible behavior changes in the PR description
- Include screenshots when changing menu bar UI or settings UI

## Issues

- Use bug reports for broken behavior
- Use feature requests for new ideas or workflow improvements
- Include reproduction steps, expected behavior, and actual behavior

## Scope

This repository is for the macOS menu bar app. The marketing site and project website live in separate repositories.
